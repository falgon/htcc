newtype Ref a Source #

The type that specifies that register values are considered address values. e.g.:

>>> Ref rax
[rax]
>>> Ref rsp
[rsp]
>>> import qualified Data.Text as T
>>> T.putStr $ mov rax (Ref rsp) <> add rsp 8
     mov rax, [rsp]
     add rsp, 8
